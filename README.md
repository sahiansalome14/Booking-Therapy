
# 🌿✨ Booking Therapy - Vis Vitalis


Conectando el bienestar alternativo con la tecnología

Vis Vitalis es una plataforma digital que conecta terapeutas independientes del sector salud alternativa con clientes que buscan servicios de bienestar confiables y centralizados. Un ecosistema B2C/B2B especializado que facilita la reserva de sesiones, la gestión profesional y el acceso a productos asociados a las terapias.

## 🎯 Nuestro Enfoque

### Mercado Vertical

Salud y bienestar alternativo, priorizando:

* Terapeutas independientes certificados: acupunturistas, fisioterapeutas, psicólogos clínicos, masajistas terapéuticos

### Modelo de Negocio

Ecosistema dual que conecta:

🧑‍⚕️ Profesionales de la salud alternativa

👥 Pacientes / clientes finales

🛍️ Productos asociados a las terapias



## 📁 Estructura del proyecto

Proyecto dividido en backend (Django) y frontend (React + Vite).

```bash

│
├── backend/          # API REST con Django + DRF

│
├── frontend/         # Aplicación React con Vite
│   └── README.md     # Instalación y configuración del frontend
│
└── README.md         # Este archivo

```

## 🚀 Inicio rápido


Asegúrate de tener instalado:

* Python 3.10+ & Poetry

* Bun (o Node.js) o npm

* PostgreSQL

```bash
# Ver instrucciones detalladas en backend/README.md

```


Backend: Django, Django REST Framework, Poetry
